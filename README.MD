# ru
このソフトを利用することで誰でもルー大柴になれると言っても過言ではないように思う


## ミーのルーをユーのパーソナルなコンピュータでエクセキューションするメソッドゥ
```sh
git clone "https://github.com/CP-WD/ru_generator.git"
cd ru_generator
pip install -r requirements.txt
python util/fetch_dict.py
python main.py
```


## ミーのルーのユーズのイグザンプル
一緒にしようぜ
-> トゥギャザーにしようぜ

馬鹿も休み休み
-> フールも休み休み

雨降って地固まる
-> レイン降ってアース固まる
